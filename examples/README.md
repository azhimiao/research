# Research examples

4 个 skill，与 `skill-forge/built/research/<skill>` 同构。

```bash
python skill-core/skill.py install examples/research/research-survey --host cursor --scope global
```

- `literature-survey` — 检索与验证
- `paper-quality-gate` — 质量审计
- `research-survey` — 入口编排（推荐）
- `review-synthesis` — 叙事综述

重新导出：`python skill-core/scripts/export_examples.py`（需完整 skill-forge/built 树）
